# Online Banking Mini System

## Overview

A console-based banking application developed using Java, JDBC, and MySQL. The system provides secure user authentication, balance inquiry, deposit, withdrawal, and transaction history functionalities.

## Technologies Used

* Java
* JDBC
* MySQL
* Git & GitHub

## Features

* User Login Authentication
* Balance Inquiry
* Deposit Money
* Withdraw Money
* Transaction History
* DAO-Service-UI Layered Architecture

## Project Structure

* UI Layer
* Service Layer
* DAO Layer
* MySQL Database
